Our masterclass

/// What you will learn

In this masterclass, students will master virtual production techniques using Aximmetry and Unreal Engine. They will learn fundamentals like setting up environments, working with cameras, lights, and materials. The course covers real-world applications, such as live streaming, while providing expert tips and project management skills to create professional-quality virtual productions.

How to get proper key in Aximmetry:

Users will learn how to harness the power of Unreal Engine and Aximmetry to create real-time virtual environments, visual effects, and interactive experiences, enabling them to produce captivating content for various industries such as gaming, film, and live events.

Advanced Techniques and Tools:

Participants will gain in-depth knowledge of advanced features and techniques in both Unreal Engine and Aximmetry, including dynamic materials, dynamic lighting, virtual screens, in-production controlled decals, and tracking using HTC Vive or ReTracker Bliss. This will allow them to create more sophisticated and immersive projects.

Workflow Optimization and Integration:

Users will discover how to optimize their workflows using both Unreal Engine and Aximmetry effectively, as well as integrate these tools with other software and hardware. This includes learning about automation, scripting, external control, and leveraging available plugins and libraries for a seamless and efficient production process.
